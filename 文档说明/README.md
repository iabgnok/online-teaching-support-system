## 💡 重要提示

### 关于权限系统
系统最终选择了简单的权限级别设计（1-3级），而不是复杂的细粒度权限系统。相关文档：
- [新权限系统尝试记录.md](新权限系统尝试记录.md) - 说明了为何选择简单设计

### 关于数据库设计
数据库设计符合 BCNF 范式，详见：
- [数据库规范化分析报告.md](数据库规范化分析报告.md)
